Privacy Policy

Last updated: June 30, 2026

Overview

Findora is designed as a privacy-first Android file management application. The app helps users search, organize, analyze, and manage files stored locally on their device.

Findora operates primarily offline. An internet connection is not required for core file management features.

Local File Processing

Findora processes files locally on the device. File names, file paths, file metadata, file categories, duplicate detection results, storage summaries, APK information, screenshots, downloads, and large file analysis are handled on the user's Android device.

No files are uploaded to any server. All file processing is performed locally on your device.

Personal Information

Findora does not collect personal information. The app does not require your name, email address, phone number, account profile, location, contacts, or other personal identifiers.

No user accounts are required to use Findora.

Analytics and Advertising

No analytics services are used in Findora. The app does not include tracking tools that monitor user behavior for analytics reporting.

No advertising SDKs are included in Findora. The app does not use ad networks, advertising identifiers, or advertising-based tracking.

File Permissions

Depending on the Android version, Findora may request storage or file access permissions, including the All Files Access permission where applicable, solely to provide file management features.

These permissions are used only to search, organize, analyze, and manage files stored locally on your device.

Features that rely on these permissions include Smart File Search, Download Management, Duplicate File Detection, Screenshot Management, APK Management, Large File Analysis, and Storage Insights.

Permission Control

Users can revoke permissions at any time from Android Settings. If permissions are revoked, some Findora features may stop working because the app will no longer be able to access the local files needed for those features.

Data Sharing

Because Findora does not collect personal information and does not upload files to any server, Findora does not sell, rent, trade, or share personal information with third parties.

Security

Findora processes files locally on your device. We take reasonable measures to ensure that no file content is transmitted outside your device during normal operation.

Third-Party Services

Findora does not rely on third-party cloud services for file processing. If future versions introduce optional third-party services, this Privacy Policy will be updated accordingly.

Children's Privacy

Findora is not directed to children and does not knowingly collect personal information from children.

Changes to This Policy

This Privacy Policy may be updated from time to time. Any changes will be posted on this page with an updated date.
